# 마우스 길들이기란?

   * 특정 위치를 저장하여 커서를 분실했을 시 그 위치로 오도록 하거나, 현재 위치를 알려줍니다. 
   * 초당 n번 클릭이 가능한 오토클릭 기능을 제공합니다.
   * 일정 시간 뒤에 마우스가 숨김 상태가 되게하는 기능을 제공합니다
   * 커서의 크기를 조절할 수 있습니다.
   * 기본 모양, 대기 모양, 버튼 모양의 커서를 원하는 파일로 쉽게 변경이 가능합니다.
